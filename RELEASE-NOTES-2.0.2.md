# PaketHub v2.0.2 — Search Focus Hotfix

PaketHub 2.0.2 fixes a remaining focus issue in the dashboard search field.

## Fixed

- The search cursor no longer disappears after pressing Backspace.
- Focus remains inside the PaketHub search field while typing or deleting.
- Cursor position is restored after package filtering rerenders the card.
- Home Assistant global keyboard shortcuts remain isolated from the search input.

## Upgrade

Restart Home Assistant after updating and perform a hard browser refresh.
