# PaketHub v2.0.1 — Search Input Fix

PaketHub 2.0.1 is a focused maintenance update for the dashboard search field.

## Fixed

- Home Assistant's global search no longer opens while typing in the PaketHub package search.
- The PaketHub search field retains focus after every character.
- The text cursor remains in the correct position while package results are filtered.

## Upgrade

Restart Home Assistant after updating and perform a hard browser refresh (`Ctrl + F5` or `Cmd + Shift + R`).
