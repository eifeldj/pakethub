# PaketHub v2.3.1

PaketHub 2.3.1 is a small branding and presentation update.

## Fixed

- Added the missing PaketHub icon for HACS.
- Added a high-resolution icon for sharp display on high-density screens.
- Improved the custom-card metadata used by the Home Assistant card picker.
- Added a direct documentation link to the card editor metadata.

## Compatibility

- Home Assistant 2025.1 or newer
- No migration required

After updating, refresh the HACS repository information, restart Home Assistant, and perform a hard browser refresh.
