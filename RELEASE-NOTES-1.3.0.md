# PaketHub v2.0.1 — Readable Cards & Responsive Details

PaketHub 2.0.1 focuses on the real-world readability of shipment cards, especially inside narrow Home Assistant dashboard columns.

## Highlights

- Full package names with multi-line wrapping
- Full latest tracking status without clipping
- Relative time below the latest status
- Matching ETA and location tiles
- More space for shipment information
- Container-query responsive layout
- Improved tablet, narrow-column and mobile presentation
- Existing carrier branding, keyboard support and accessible detail dialog retained

## Upgrade

Restart Home Assistant after updating and perform a hard refresh of the browser.
