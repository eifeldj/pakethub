# PaketHub v2.2.0

PaketHub 2.2.0 introduces a redesigned inline tracking timeline that keeps shipment details directly inside the dashboard.

## New

- Expandable tracking history inside every package card
- Compact view showing the three newest tracking events
- “Show all events” and “Show fewer” controls for longer histories
- Separate display of event time, location and tracking provider
- Clear highlighting of the newest tracking event
- Smooth expansion animation
- Improved mobile layout and accessibility

## Compatibility

- Home Assistant 2025.6+
- No migration required from PaketHub 2.1.x

After updating, restart Home Assistant and perform a hard browser refresh.
