# Changelog

All notable changes to PaketHub are documented in this file.

## 1.0.0

- First stable PaketHub release
- Added the native PaketHub package detail dialog
- Added a chronological tracking timeline with date, time, status, location, and carrier
- Highlighted the most recent tracking event
- Added copy-to-clipboard support for tracking numbers
- Added direct links to 17TRACK and Home Assistant entity details
- Added the native Lovelace card with visual editor support
- Added shipment counters, status chips, progress bars, ETA display, and manual refresh
- Added PaketHub branding, icon, and HACS banner
- Added English project documentation and a separate German README
- Unified integration, frontend, and Lovelace resource versions at 1.0.0
- Cleaned release files of caches, temporary files, and platform metadata

## 0.9.1

- Unified integration, dashboard, and frontend resource versions
- Updated the Lovelace resource cache-busting URL
- Packaged branding files, documentation, and translations

## 0.9.0

- Added the package detail view and tracking timeline
- Added tracking-number copy support and the direct 17TRACK link
- Added new PaketHub branding

## 0.8.1

- Fixed frontend cache invalidation and version consistency

## 0.8.0

- Added the smart header, package summary, status chips, progress bars, ETA display, and responsive layout

## 0.7.0

- Added the native `custom:pakethub-card` Lovelace card and visual editor
