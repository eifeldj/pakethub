# PaketHub v2.1.1 — Delivered Package Archive

PaketHub 2.1.1 makes completed shipments accessible through a dedicated delivered-package view.

## New

- Status filters for **All**, **Active**, **In transit**, **Delivered**, **Problems** and **Favorites**
- Delivered package counter in the filter toolbar
- On-demand archive view without cluttering the active package list
- Search works inside every filter, including delivered shipments
- Responsive filter layout for desktop, tablet and mobile

No configuration migration is required.
