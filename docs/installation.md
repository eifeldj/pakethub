# Installation

## HACS

1. Open **HACS**.
2. Add `https://github.com/eifeldj/pakethub` as a custom **Integration** repository.
3. Install PaketHub.
4. Restart Home Assistant.
5. Add PaketHub under **Settings → Devices & services**.
6. Enter the official 17TRACK API v2.4 key.

## Manual installation

Copy `custom_components/pakethub` to `/config/custom_components/pakethub`, restart Home Assistant and add the integration.
