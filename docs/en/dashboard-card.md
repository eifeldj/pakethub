# Dashboard card

```yaml
type: custom:pakethub-card
title: My packages
show_delivered: false
max_packages: 8
sort_by: status
tap_action: details
```

## Tap actions

- `details` — PaketHub detail dialog
- `url` — shipment page on 17TRACK
- `more-info` — Home Assistant entity dialog

The JavaScript resource is registered automatically for storage-mode dashboards.
