# Roadmap

The roadmap describes direction, not a guaranteed delivery schedule.

## Current foundation

- [x] Provider framework
- [x] 17TRACK registry and fallback
- [x] Native UPS tracking
- [x] Provider diagnostics and runtime metrics
- [x] Bilingual documentation
- [x] Automated GitHub Pages deployment

## Next candidates

- [ ] Additional native carriers
- [ ] Smarter carrier confidence scoring
- [ ] Provider response caching
- [ ] Improved dashboard filtering and grouping
- [ ] Home Assistant events for important shipment changes
- [ ] Public provider-development API

## Long-term direction

PaketHub should remain simple for users while allowing carrier-specific tracking implementations to evolve independently.
