# Features

## Shipment management

- Add, update and remove tracked shipments
- One Home Assistant device per shipment
- Consistent entities and attributes
- Human-readable status, ETA, progress and latest location

## Provider intelligence

- Automatic carrier detection
- Native provider preferred where available
- Graceful fallback to 17TRACK
- Normalized tracking data independent of the source

## Home Assistant experience

- Dedicated PaketHub dashboard card
- Service/action support for automations
- Diagnostics suitable for troubleshooting
- Integration and frontend version synchronization

## Operational visibility

- API-call and runtime statistics
- Provider usage and fallback counters
- Last successful refresh and update duration
- Structured logging
