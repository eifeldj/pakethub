# Actions and automations

## Add a package

```yaml
action: pakethub.add_package
data:
  tracking_number: EXAMPLE1234567890
  package_name: Example order
```

## Remove a package

```yaml
action: pakethub.remove_package
data:
  tracking_number: EXAMPLE1234567890
```

## Refresh

```yaml
action: pakethub.refresh
```
