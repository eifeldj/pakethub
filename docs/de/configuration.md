# Konfiguration

PaketHub wird über die Home-Assistant-Oberfläche konfiguriert.

## Erforderlich

- Offizieller 17TRACK-API-Schlüssel v2.4

## Abfrageintervall

Wähle ein angemessenes Intervall, um unnötige API-Aufrufe und Carrier-Limits zu vermeiden.

## Unterstütztes Setup

Aktuell wird ein 17TRACK-Konto pro Home-Assistant-Instanz unterstützt.
