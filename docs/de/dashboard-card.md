# Dashboard-Karte

```yaml
type: custom:pakethub-card
title: Meine Pakete
show_delivered: false
max_packages: 8
sort_by: status
tap_action: details
```

## Tipp-Aktionen

- `details` — PaketHub-Detaildialog
- `url` — Sendungsseite bei 17TRACK
- `more-info` — Home-Assistant-Entitätsdialog

Die JavaScript-Ressource wird bei Dashboards im Storage-Modus automatisch registriert.
