# Roadmap

Die Roadmap beschreibt die Richtung und ist kein verbindlicher Liefertermin.

## Aktuelle Grundlage

- [x] Provider-Framework
- [x] 17TRACK als Register und Fallback
- [x] Natives UPS-Tracking
- [x] Provider-Diagnose und Laufzeitmetriken
- [x] Zweisprachige Dokumentation
- [x] Automatische Veröffentlichung über GitHub Pages

## Nächste Kandidaten

- [ ] Weitere native Carrier
- [ ] Intelligentere Carrier-Erkennung mit Vertrauenswert
- [ ] Zwischenspeicherung von Provider-Antworten
- [ ] Verbesserte Filterung und Gruppierung im Dashboard
- [ ] Home-Assistant-Events bei wichtigen Sendungsänderungen
- [ ] Öffentliche API zur Provider-Entwicklung

## Langfristige Richtung

PaketHub soll für Anwender einfach bleiben, während carrierspezifische Tracking-Implementierungen unabhängig weiterentwickelt werden können.
