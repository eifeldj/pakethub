# PaketHub v1.3.1

A focused visual refinement release for the redesigned package cards.

## Changes
- Removed duplicate carrier labels for wordmark-style logos such as YunExpress.
- Improved carrier logo proportions and alignment.
- Limited long package names to two lines.
- Reduced status badge prominence.
- Strengthened progress bars with a subtle glow.
- Added understated carrier-colour accents to package cards.
