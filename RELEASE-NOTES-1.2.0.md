# PaketHub v1.2.0 — Carrier Branding & Spacious Layout

PaketHub 1.2.0 introduces a redesigned package overview focused on carrier recognition, readability and responsive use.

## Highlights

- Dedicated carrier brand panel using recognizable carrier colors
- New wide card layout with more space at the right edge
- Larger package title, progress bar, ETA and location details
- Compact carrier header on mobile devices
- Built-in branding for Austrian Post, DHL, UPS, FedEx, GLS, DPD, Amazon Logistics, YunExpress, Hermes/Evri, Royal Mail and USPS
- Existing accessible detail dialog and keyboard navigation retained

## Upgrade

Replace the existing repository files with the contents of this archive, restart Home Assistant, and perform a hard refresh of the browser.
