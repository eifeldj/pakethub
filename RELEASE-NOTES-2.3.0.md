# PaketHub v2.3.0

PaketHub 2.3.0 makes the dashboard overview interactive and adds more flexible shipment sorting.

## Interactive overview

- Click **Active**, **In transit**, **Delivered**, or **Problems** to filter the package list immediately
- The selected overview tile is visually highlighted
- Overview tiles remain synchronized with the existing status filters

## Advanced sorting

Shipments can now be sorted directly in the dashboard by:

- Status
- Delivery date
- Last update
- Name
- Carrier

The new options are also available in the visual card editor.

## Additional improvements

- Responsive mobile layout for the sort control
- Improved keyboard accessibility for overview controls
- Updated documentation and version references

No migration is required. Restart Home Assistant and perform a hard browser refresh after updating.
